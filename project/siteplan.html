AuraScents Perfume Store

Website Planning Document

Site Name

AuraScents Perfume Store

The name was chosen because it sounds modern and elegant and represents a variety of perfumes, including fresh, sweet, woody, and oriental fragrances.

Optional domain: aurascents.com

Site Purpose

The purpose of this website is to present an informational perfume store that allows users to explore fragrances by category and style. The site helps users learn about perfumes based on season, scent type, and personal preference. It also simulates an e-commerce layout without real online purchases.

Scenarios

  • What type of perfume is best for summer or warm weather?
  • Where can I find oriental or Arabic perfumes?
  • Are there unisex perfumes available?

Color Schema

  • Dark Charcoal (#1c1c1c): Used for headers, navigation menu, and footer.
  • Soft Gold (#c9a24d): Used for buttons, links, and accent elements.
  • Light Beige (#f5f3ef): Used as the main background color.

Typography

  • Playfair Display: Used for headings and titles.
  • Roboto: Used for body text and paragraphs.
